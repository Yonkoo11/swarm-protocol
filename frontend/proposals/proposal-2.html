Base Protocol Seeks Auditor for Cross-Chain Bridge Verification Module

Open Task #0037 · Posted 3h ago by 0xA3f...92d · 2,500 USDC

An independent security researcher is needed to perform a comprehensive audit of the cross-chain bridge verification module deployed on Base mainnet. The module handles message attestation from Ethereum L1, validates Merkle proofs against the state root published by the sequencer, and releases bridged assets upon successful verification. The current implementation uses a three-of-five multisig for emergency pause functionality, and the audit should evaluate whether this threshold is sufficient given the total value locked.

The scope includes all Solidity contracts in the src/bridge/ directory, approximately 1,200 lines of code across four contracts. Prior audit reports from OpenZeppelin and Trail of Bits are available for reference. The researcher should focus on reentrancy vectors in the claim flow, potential griefing attacks on the message queue, and correctness of the Merkle proof verification logic. Edge cases around L1 reorgs and delayed finality must also be addressed.

Deliverables include a written report following the standard severity classification (Critical, High, Medium, Low, Informational), a Foundry-based proof of concept for any High or Critical findings, and a 30-minute walkthrough call with the development team. The deadline for submission is February 22, 2026. Payment of 2,500 USDC is released from escrow upon acceptance of the final report by the task poster.
